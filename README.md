## About

**Faham Remake** *is a full-stack web application for managing your University courses, assignments, and grades. The app leverages modern web technologies to provide fast performance, secure authentication, and a responsive, user-friendly UI.*  

*This is project is still in development, but can be previewed [here](https://fahamremake.vercel.app/)*

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/ZashShadow/faham.git
   cd faham
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

## Not Currently Needed
3. **Configure environment variables**  
   Create a `.env.local` file based on `.env.example` and add your values for MongoDB, Auth.js, and Gemini API configuration.

4. **Run the development server**
   ```bash
   npm run dev
   ```
   Then go to `http://localhost:3000` in your browser.

## Usage
1. Register or sign in using the Auth.js authentication flow.
2. Preview your courses, assignments, and grades via the user dashboard.
3. Download FEE Chalan.
